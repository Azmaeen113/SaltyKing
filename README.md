# Salty Kingdom

Web mini-game prototype built as a single-page HTML app.

## Project Structure

- `index.html` - main game (all UI screens and game logic)
- `dicerolling.html` - standalone dice demo/reference
- `salty.png` - main board background image
- `splash screen.png` - splash background image
- `logo/` - social icons and video assets used by the UI

## Local Run

This is a static site. Open `index.html` directly, or run any static server.

## Deploy

Configured for Netlify static deploy using:

- `netlify.toml`

Default publish directory is project root (`.`).
